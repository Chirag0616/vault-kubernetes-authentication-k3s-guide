# HCP Vault Kubernetes Authentication Setup and Troubleshooting Guide (K3s)

**Author:** Chirag Jain

## Overview

This repository contains a detailed implementation and troubleshooting guide for integrating HCP Vault Kubernetes Authentication with a K3s Kubernetes cluster.

The guide covers complete setup, validation, troubleshooting, and resolution of common authentication issues, including Kubernetes TokenReview permissions, TLS SAN configuration, and 403 Permission Denied errors.

## Contents

* Service Account Creation
* TokenReview Permissions
* Kubernetes CA Certificate Configuration
* Kubernetes API Endpoint Validation
* HCP Vault Kubernetes Authentication Configuration
* Vault Policy Configuration
* Vault Kubernetes Role Configuration
* TLS SAN Validation for K3s API Server
* Kubernetes JWT Authentication Testing
* Troubleshooting 403 Permission Denied Errors
